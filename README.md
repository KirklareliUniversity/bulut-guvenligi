Bulut Güvenliği: Azure ve AWS Platformlarında Kapsamlı Güvenlik Analizi
Proje Hakkında
Bu proje, bulut tabanlı sistemlerde güvenliğin sağlanması, modern tehditlerin analizi ve Microsoft Azure ile Amazon Web Services (AWS) platformlarında kullanılan güvenlik mekanizmalarının karşılaştırmalı olarak incelenmesi amacıyla hazırlanmıştır. Çalışma, 2025 yılı itibarıyla artan bulut tehditlerini, yanlış yapılandırma risklerini, kimlik yönetimi zafiyetlerini ve yapay zekâ destekli saldırıları kapsamlı biçimde ele almaktadır.
Proje; teorik bilgiler, güncel vaka analizleri, gerçek senaryolar ve en iyi uygulamalar üzerinden ilerleyerek, bulut güvenliğine bütüncül ve profesyonel bir bakış açısı sunmayı hedefler.

Projenin Amacı
•	Bulut güvenliği tehditlerini ve güncel trendleri analiz etmek
•	Azure ve AWS platformlarında kullanılan güvenlik mekanizmalarını detaylı incelemek
•	Kimlik doğrulama, yetkilendirme, ağ güvenliği ve veri koruma süreçlerini açıklamak
•	Yanlış yapılandırmaların ve zayıf güvenlik politikalarının doğurduğu riskleri ortaya koymak
•	Kurumlar için uygulanabilir güvenlik stratejileri ve çözüm önerileri sunmak

Proje Kapsamı
Bu proje aşağıdaki ana başlıkları kapsamaktadır:
•	Bulut güvenliği tehditleri (2025 perspektifi)
•	Kimlik doğrulama ve yetkilendirme (IAM, RBAC, MFA)
•	Yanlış yapılandırmalar ve veri ihlalleri
•	Yapay zekâ destekli saldırılar ve ransomware tehditleri
•	OAuth, API ve token güvenliği
•	Azure ve AWS güvenlik servislerinin karşılaştırılması
•	IDS / IPS sistemleri
•	İzleme, loglama ve SIEM entegrasyonu
•	DevSecOps ve CI/CD güvenlik yaklaşımları
•	Gerçekçi Azure ve AWS erişim senaryoları

Kullanılan Platformlar ve Teknolojiler
•	Microsoft Azure
o	Azure Active Directory (AAD)
o	RBAC (Role Based Access Control)
o	Azure Policy
o	Azure Monitor & Defender for Cloud
o	Azure Key Vault
o	Azure Firewall & DDoS Protection
•	Amazon Web Services (AWS)
o	AWS IAM
o	IAM Policies
o	AWS Config
o	CloudTrail & CloudWatch
o	AWS KMS
o	GuardDuty, WAF ve Shield

Metodoloji

•	En iyi uygulamaların (Best Practices) değerlendirilmesi

•	DevSecOps yaklaşımına yönelik yol haritası

Hedef Kitle
